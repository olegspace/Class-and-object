Задания по теме «Классы и объекты»
1. Создай класс «слайм» и у него следующие поля — имя, размер, вес, цвет. Также два метода — представиться (написать «Привет, меня зовут <имя>» и испугаться — как параметр метод «испугаться» получает другого слайма и выясняет его размер, сравнивает. Если размер другого слайма больше, возвращает True, иначе False).
После этого создайте двух слаймов и пусть они представятся, но вторым представится тот, кто испугался, а первым тот, кто нет. Используйте в программе оба метода.
2. Создать класс NPC, содержащие: имя, рост, здоровье, профессия, список товаров и услуг. Задай классу методы приветствия, использующие имя NPC, прощания и предложения услуг. Создай три NPC с разными профессиями и списками товаров/услуг
3. Создай структуру с именем student, содержащую поля: фамилия и инициалы, номер группы, успеваемость (набор из пяти элементов). Создать список из десяти элементов такого типа. Добавить возможность вывода фамилий и номеров групп студентов, имеющих успеваемость только выше 4 или только ниже 4.
4. Создай структуру с именем train (поезд), содержащую поля: название пункта назначения, номер поезда, время отправления. Ввести данные в массив из пяти элементов типа train, упорядочить элементы по номерам поездов. Добавить возможность вывода информации о поезде, номер которого введен пользователем.
5. Менеджер задач, где каждая задача — объект. У объекта есть поля «задача», «важность»(высокая/средняя/низкая), «дедлайн»(к какой дате выполнить), «категория»(дом/школа/хобби и т.д.). В цикле while вводим команду. Есть команда поиск, добавить, удалить,редактировать задачу, вывести все задачи.
